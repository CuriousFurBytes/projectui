# Similar Projects & Inspiration

ProjecTUI was inspired by these tools and projects:

| Project | Notes |
|---------|-------|
| [ascii-motion.app](https://ascii-motion.app/) | ASCII art animation editor in the browser. Inspired the idea of a canvas-based terminal design tool. |
| [tui.builders](https://tui.builders) | Another web-based TUI design tool. Direct peer in the same problem space. |
| [Lazygit](https://github.com/jesseduffield/lazygit) | A beautifully designed TUI app in Go. Exemplary use of Bubble Tea-style layout — a major design reference for what good TUI design looks like. |
| [xterm.js](https://xtermjs.org/) | Browser-based terminal emulator. Shows how real terminals render in the browser. |
| [ttyd](https://github.com/tsl0922/ttyd) | Share your terminal over the web. Good reference for in-browser terminal rendering. |
| [VHS](https://github.com/charmbracelet/vhs) | Record your terminal as a GIF. Another Charm tool showing how terminal UIs can be captured and shared. |
| Figma for Terminal (concept) | Community discussion threads about what a visual TUI design tool should look like. |

::: tip
Know another TUI builder or ASCII canvas tool? [Open an issue](https://github.com/lucasqueiroz/projectui/issues) and we'll add it to the list.
:::
