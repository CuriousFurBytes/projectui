# Sharing Projects

Click **Share** in the top bar to generate a URL that encodes your entire project in the URL fragment. The project JSON is compressed (LZ-string) and Base64-encoded. Recipients open the URL and the project loads directly — no files exchanged, no login required.

::: warning Long URLs
Share URLs can get long for large projects. Most browsers and URL shorteners handle them fine, but some chat tools truncate long URLs. Use **Export JSON** for large projects that need reliable sharing.
:::
