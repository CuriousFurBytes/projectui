# Layers & Screens

A **project** contains one or more **screens** (also called layers). Each screen is an independent canvas with its own component tree. Use multiple screens to design an entire application — login, dashboard, detail view, modal, etc.

## Managing Screens

- Click **+ Screen** in the Layers panel to add a new screen.
- Double-click a screen tab to rename it.
- Drag a tab to reorder screens.
- Click the **eye icon** to toggle visibility (hidden screens are still exported).
- Click the **lock icon** to prevent accidental edits.
- The **trash icon** deletes a screen.

## Component Tree

The Layers panel also shows the **component tree** for the active screen. You can drag items in the tree to rearrange nesting, and click any node to select it on the canvas.

::: tip Wiring screens together
Once you have multiple screens, use the [Timeline panel](/usage/timeline) to model how a user navigates between them.
:::
