# Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| <kbd>⌘Z</kbd> / <kbd>Ctrl+Z</kbd> | Undo |
| <kbd>⌘⇧Z</kbd> / <kbd>Ctrl+Shift+Z</kbd> | Redo |
| <kbd>⌫</kbd> / <kbd>Delete</kbd> | Delete selected component |
| <kbd>⌘K</kbd> / <kbd>Ctrl+K</kbd> | Open command palette |
| <kbd>⌘</kbd> + click | Multi-select components |
| <kbd>Escape</kbd> | Deselect / close palette |
